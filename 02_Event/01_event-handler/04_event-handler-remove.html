04. 이벤트 핸들러 제거

        EventTarget.prototype.removeEventListener 메소드를 통해
        addEventListener 메소드로 등록한 이벤트 핸들러를 제거한다.
        removeEventListener 메소드에 전달할 인수는 addEventListener
        이벤트 핸들러는 제거되지 않는다.
    
        프로퍼티 방식으로 등록한 이벤트 핸들러 제거 방식
        removeEventListener 메소드로 제거를 할 수 없다.
        null값을 넣으면 된다.
